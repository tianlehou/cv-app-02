Datos de Contacto & Personales

¿Qué datos personales incluir en tu CV?

  • Nombre completo:Tu nombre tal como deseas que te conozcan profesionalmente.
  • Ubicación:Ciudad y país. La dirección exacta no es necesaria.
  • Email profesional:Evita correos informales como chico_fiesta@...".
  • Teléfono:Incluye código de área si aplica.
  • Opcionales:Edad, nacionalidad o estado civil (si lo requiere el país o sector).
  • Evita:Número de documento, dirección completa, creencias, fotos familiares.
